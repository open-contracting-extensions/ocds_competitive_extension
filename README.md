# Competitive

Adds a field to the tender object in a framework agreement call-off to indicate if it is competitive or direct.

## Examples

### Framework agreement direct call-off

  "tender": {
    "id": "2421-1016-CM20",
    "procuringEntity": {

    "competitive": false
  },
  "awards": [

      "id": "42133251",
